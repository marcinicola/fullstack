# Closures

Scrivi una nuova `chiusura ': crea una funzione chiamata` multiplybytwo` che assume un parametro.
All'interno della funzione `multiplybytwo`, crea una funzione chiamata` interiore '.Questa funzione si occuperà di moltiplicare il parametro passato come input e moltiplicarlo per la variabile denominata `numero`
Stampa nella console l'output della funzione.

Esempio: `multiplybytwo (4) ()`, risultato: `8`
